# 📱 5G 随身WiFi iOS 应用

一键通过 GitHub Actions 编译，无需 Mac！

---

## 🚀 快速开始

1. 点击顶部的 "Actions" 选项卡
2. 点击左侧的 "Build 5GMobileWiFi IPA"
3. 点击右侧的 "Run workflow" 按钮
4. 等待 3-5 分钟
5. 在页面底部下载 IPA 文件

---

## 📲 安装到手机

1. 下载 Sideloadly：https://sideloadly.io/
2. 连接 iPhone 到电脑
3. 把下载的 IPA 拖进 Sideloadly
4. 输入你的 Apple ID（免费的就行）
5. 点击 "Start"

第一次打开需要在 iPhone 设置里信任证书！

---

## 💡 功能说明

- ✅ 主页 - 显示设备状态、信号、流量等
- ✅ 短信 - 收发和转发短信
- ✅ 网络 - 移动网络、DNS、VPN 等设置
- ✅ WiFi - 2.4G/5G WiFi 设置
- ✅ 设置 - 系统管理、设备连接
